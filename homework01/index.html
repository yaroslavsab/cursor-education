Шпаргалка по GIT!

  • git config --global user.name blabla - команда яка добавляє в глобальний конфіг користувача з імям blabla
  • git config --global user.email blabal@bla - команда яка добавляє в глобальний конфіг користувача з email blabal@bla
  • git init - команда яка встановлює git в поточну директорію
  • git status - показує статус роботи git
  • git add - добавляє файл або всі файли в staged
  • git commit - файли з staged фіксує в локальний репозиторій, в лапках вказується короткий опис змін
  • git log - історія версій поточної гілки
  • git checkout - використовується для переключення гілок
  • git branch - команда для керування гілками
  • git push - надсилання змін до відділеного репозиторію
  • git pull - отримання змін з відділеного репозиторію
  • git merge - зливання однієї чи більше гілок до пточної
  • git clone - копія GIT репозиторію
  • файл .gitignore - файл містить список файлів які будуть ігноруватись git'ом
